# ProteinWatch

A professional, premium-grade real-time early warning system for tracking viral protein threats globally. 

## Features

- **Sequence Analyzer**: Analyze viral sequence novelty, danger classification, and structural scores.
- **Global Threat Map**: Interactive full-screen map highlighting threat signals globally.
- **Agent Trace Timeline**: View steps taken by underlying ingestion, fingerprinting, and containment agents.
- **Crisis Briefings**: Token-by-token streaming AI intelligence briefs in multiple languages (English & Urdu).
- **Simulate Action & Alerts**: Initiate quarantine protocols and generate immutable alert tickets.
- **3D Protein Viewer**: Interactive structural folding inspection.
